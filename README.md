# Superstore Sales Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Superstore dataset using Python, Pandas, and Matplotlib. The goal is to analyze sales performance, profitability, regional trends, product performance, and the impact of discounts on profit.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Git
- GitHub

## Dataset
The project uses the Superstore retail dataset containing customer orders, sales, profit, discounts, products, and regional information.

## Analysis Performed
- Total Sales Analysis
- Total Profit Analysis
- Sales by Category
- Profit by Category
- Sales by Region
- Top 10 Products by Sales
- Monthly Sales Trend Analysis
- Discount vs Profit Analysis

## Key Insights
- Technology category generated the highest sales and profit.
- West region recorded the highest overall sales.
- November was the highest sales month.
- Canon imageCLASS 2200 Advanced Copier was the top-selling product.
- Higher discounts often resulted in lower profits.
